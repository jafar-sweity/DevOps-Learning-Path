| pwd     | Prints the current working directory | pwd |  |
| man     | Displays the manual page for a command | man ls | Exit the manual using q |
| clear   | Clears the terminal screen | clear |  |
| whoami  | Displays the effective user name | whoami |  |
| echo    | Prints text or variable values | echo $PATH |  |
| history | Shows a list of previously executed commands | history |  |
