# Package Management (DEB/APT)

| Command     | Description                      | Example Usage         | Notes                                 |
| ----------- | -------------------------------- | --------------------- | ------------------------------------- |
| apt update  | Synchronizes package index files | sudo apt update       | Essential before installing/upgrading |
| apt upgrade | Installs updated packages        | sudo apt upgrade      |                                       |
| apt install | Installs new packages            | sudo apt install htop | Handles dependencies automatically    |
| apt search  | Searches for packages            | apt search editor     |                                       |
