# File Creation, Deletion, & Management

| Command | Description                                | Example Usage                | Notes |
| ------- | ------------------------------------------ | ---------------------------- | ----- |
| touch   | Creates an empty file or updates timestamp | touch file1.txt              |       |
| rm      | Removes (deletes) files                    | rm file1.txt                 |       |
| rm -r   | Recursively removes directories            | rm -r my_folder              |       |
| mkdir   | Creates a new directory                    | mkdir new_dir                |       |
| cp      | Copies files or directories                | cp source.txt /tmp/          |       |
| mv      | Moves or renames files/directories         | mv old_name.txt new_name.txt |       |
| ln -s   | Creates a symbolic link                    | ln -s target link_name       |       |
