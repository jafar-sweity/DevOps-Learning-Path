# Permissions Exercises

## Exercise 1: Change file permissions

- Command: `chmod 755 script.sh`
- Task: Set execute permission for owner and read for others.

## Exercise 2: Change file ownership

- Command: `sudo chown user:group file.txt`
- Task: Assign a new owner and group to a file.

## Exercise 3: Switch user

- Command: `su - root`
- Task: Switch to root user.

## Exercise 4: Use sudo

- Command: `sudo apt update`
- Task: Execute a command with elevated privileges.
