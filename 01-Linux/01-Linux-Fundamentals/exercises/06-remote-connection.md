# Remote Connection Exercises

## Exercise 1: SSH into a server

- Command: `ssh user@localhost`
- Task: Connect to a remote server via SSH.
- Try to connect to docker container \& my own laptop also

## Exercise 2: Copy files with SCP

- Command: `scp local.txt user@server:/tmp/`
- Task: Transfer a local file to a remote host.

## Exercise 3: Sync files with rsync

- Command: `rsync -avz local/ user@remote:/data/`
- Task: Efficiently sync directories between hosts.
