# Text Filtering Exercises

## Exercise 1: View file content

- Command: `cat /etc/hosts`
- Task: Display content of `/etc/hosts`.

## Exercise 2: Search text in files

- Command: `grep 'localhost' /etc/hosts`
- Task: Find lines containing 'localhost'.

## Exercise 3: Display start/end of files

- Commands:
  - `head -n 5 /var/log/syslog`
  - `tail -n 5 /var/log/syslog`
- Task: View first and last 5 lines.

## Exercise 4: Word count

- Command: `wc -l /etc/passwd`
- Task: Count the number of lines in `/etc/passwd`.
