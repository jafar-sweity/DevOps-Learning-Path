# Linux Fundamentals Exercises

This folder contains practical exercises to apply the Linux commands and concepts learned in the cheat sheets.

## Files

1. [01-basic-navigation.md](01-basic-navigation.md) - Exercises for navigating the filesystem, checking directories, and basic commands like `ls`, `cd`, `pwd`.
2. [02-file-management.md](02-file-management.md) - Exercises for creating, copying, moving, and deleting files and directories.
3. [03-text-filtering.md](03-text-filtering.md) - Exercises for viewing, searching, and filtering file contents using commands like `cat`, `grep`, `head`, `tail`, `wc`.
4. [04-permissions.md](04-permissions.md) - Exercises for managing file permissions, ownership, and using `sudo` and `su`.
5. [05-process-management.md](05-process-management.md) - Exercises for listing, monitoring, and killing processes using `ps`, `top`, and `kill`.
6. [06-remote-connection.md](06-remote-connection.md) - Exercises for connecting to remote servers and transferring files using `ssh`, `scp`, and `rsync`.
