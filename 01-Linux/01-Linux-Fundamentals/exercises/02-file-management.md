# File Management Exercises

## Exercise 1: Create files

- Command: `touch file1.txt file2.txt`
- Task: Create two empty files in your home directory.

## Exercise 2: Create directories

- Command: `mkdir project1 project2`
- Task: Create two directories.

## Exercise 3: Copy files

- Command: `cp file1.txt project1/`
- Task: Copy `file1.txt` into `project1` directory.

## Exercise 4: Move/Rename files

- Command: `mv file2.txt project2/newname.txt`
- Task: Move and rename `file2.txt`.

## Exercise 5: Remove files/directories

- Command: `rm -r project2`
- Task: Delete the `project2` directory.
