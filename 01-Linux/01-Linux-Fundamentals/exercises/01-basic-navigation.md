# Basic Navigation Exercises

## Exercise 1: List directory contents

- Command: `ls -lha`
- Task: List all files in `/etc` including hidden files.

## Exercise 2: Change directory

- Command: `cd /var/log`
- Task: Move to `/var/log` and verify with `pwd`.

## Exercise 3: Move up one directory

- Command: `cd ..`
- Task: Go up one directory and verify path.

## Exercise 4: Check current user

- Command: `whoami`
- Task: Print your current username.
